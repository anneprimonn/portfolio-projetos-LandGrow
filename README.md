# 🌐 Site de Empresa (Projeto Acadêmico)

## 📖 Descrição
Projeto desenvolvido em equipe durante a disciplina/curso para uma empresa real.  
O site contém páginas informativas e elementos visuais para apresentar produtos/serviços.  

⚠️ Observação: Repositório não contém os arquivos reais do site por questões de confidencialidade.

## 🚀 Tecnologias
- HTML
- CSS
- PHP
- JavaScript

## 💡 Funcionalidades
- Página inicial com informações da empresa
- Questionário para identificar falhas nos setores
- Dashboard com resultados do questionário
- Agendamento de consultoria presencial
